# AR Furniture E-Commerce

## Project Overview
AR Furniture is a web-based e-commerce application that allows users to browse furniture products and preview them in augmented reality (AR) within their own space before making a purchase. The application offers a user-friendly interface for logging in, managing a shopping cart, and completing purchases, making it a comprehensive solution for online furniture shopping.

## Installation
To run this project locally, follow these steps:

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/ar-furniture-ecommerce.git
   cd ar-furniture-ecommerce
   ```

2. Open `index.html`, `products.html`, `cart.html`, or `checkout.html` in your preferred web browser.

   No additional installation is required, but make sure you have a modern browser that supports the latest web technologies.

## Usage
- **Login Page**: Users can log in using their email and password.
- **Products Page**: Browse furniture collections, preview items in AR, and add products to the shopping cart.
- **Cart Page**: Review items, adjust quantities, and view a breakdown of costs.
- **Checkout Page**: Enter payment and shipping information to finalize the purchase.

## Features
- User authentication (basic form structure provided).
- Product listing with images, descriptions, and prices.
- Augmented reality (AR) preview functionality (placeholder action).
- Shopping cart management.
- Checkout process with an order summary.

## Dependencies
This project utilizes the following libraries via CDN:
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Font Awesome**: Icon library for adding scalable vector icons.

There is no `package.json` as this project is static and does not utilize Node.js or additional packages.

## Project Structure
The project consists of the following main HTML files:

```
/
├── index.html         # Login page
├── products.html      # Product listing page
├── cart.html          # Shopping cart page
└── checkout.html      # Checkout page
```

Each HTML file is styled using Tailwind CSS and includes Font Awesome for icons.

### Style and Scripts
- Inline CSS for hover effects on product cards.
- Basic JavaScript included for AR button functionality and interaction feedback.
